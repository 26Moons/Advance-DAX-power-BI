# Power BI Mini Projects & Advanced Concepts

This repository contains small, practical Power BI projects and problem-solving exercises designed to strengthen analytical thinking, modeling skills, and DAX proficiency. Each item focuses on a specific scenario, concept, or transformation to help build deep, incremental expertise.

## Purpose
- Explore and practice advanced Power BI concepts through short examples.  
- Build strong fundamentals in data modeling, DAX, and Power Query (M).  
- Create a reference library of solutions and reusable logic.  
- Document learnings that can support future full-scale BI projects.

## What This Repository Includes

### 1. Mini Projects
End-to-end compact builds, including:
- Data cleaning & transformations  
- Star schema modeling  
- KPI and measure creation  
- Dashboard logic and aggregations  
- Real-world analytical scenarios  

### 2. Problem-Solving Exercises
Focused challenges such as:
- Custom time intelligence  
- Advanced filtering logic  
- Cumulative patterns  
- Conditional calculations  
- Relationship and context issues  

### 3. Advanced Power BI Concepts
Deep dives into:
- CALCULATE and filter context  
- Virtual tables and table functions  
- Performance tuning in DAX  
- Composite models  
- Calculation groups  
- Row-level security patterns
